# Porsche 718

This repository contains signal set configurations for the Porsche 718, organized by model year and version. The files are structured to allow for easy differentiation between model generations and other vehicle parameters, ensuring accurate signal mapping for each version of the Porsche 718.

The Porsche 718 nameplate was revived in 2016 for the fourth generation of the Boxster and Cayman models, paying homage to the successful Porsche 718 race car from the late 1950s. This generation marked a significant powertrain shift, replacing naturally aspirated flat-six engines with turbocharged flat-four engines in the base and S models, while the GTS, GT4, and Spyder variants retained naturally aspirated flat-six engines.
